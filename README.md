# 🛒 Angular Ecommerce App

A modern e-commerce web application built with Angular & TypeScript.

## ✨ Features
- Product listing & detail pages
- Reusable components
- Client-side routing
- Clean project structure

## 🛠️ Built With
- Angular 19
- TypeScript
- HTML5 / CSS3
## 🚀 Getting Started
```bash
git clone https://github.com/mostafasabrry/Ecommerce
cd Ecommerce
npm install
ng serve
```
